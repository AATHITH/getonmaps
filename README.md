# Get On Maps

A modern, responsive website for a local business visibility service that helps businesses get listed on Google Maps, JustDial, Sulekha, and provides 360° virtual tours.

## Features

- **Responsive Design**: Built with Bootstrap 5, fully responsive across all devices
- **Dark/Light Theme**: Toggle between dark and light themes for better user experience
- **Modern UI/UX**: Clean, semantic HTML with accessible design patterns
- **Testimonial Ticker**: Smooth scrolling testimonials with pause-on-hover
- **Contact Form**: Validated contact form with user-friendly error handling
- **SEO Optimized**: Proper meta tags, semantic HTML, and structured content
- **Performance**: Optimized images with lazy loading and efficient CSS/JS

## Tech Stack

- **HTML5**: Semantic markup following best practices
- **Bootstrap 5**: Modern CSS framework for responsive design
- **CSS3**: Custom styling with CSS variables for theming
- **JavaScript**: Vanilla JS for interactivity (ES6+)
- **Font Awesome**: Icon library for UI elements

## Development

The website is built with static HTML, CSS, and JavaScript. No build process is required.

To run locally:
1. Clone the repository
2. Open `index.html` in a web browser
3. For best results, use a local server (e.g., Live Server in VS Code)
